# Job Description For SAPAA App

## Opportunity

SAPAA is seeking a Multi-Platform Application Developer to continue development on a suite of applications used to support operational workflows, data retrieval, and report reading, on data about the protected sites in Alberta

## Multi-Platform Application Developer

As the new Multi-Platform Application Developer, you will work on a mobile app built with Expo, a web platform built with Next.js, and a third application developed using Google Apps Script.

You will be working with an preexisting codebase and helping develop it into a finished product. If you are inventive, committed, and experienced in modern JavaScript frameworks, this role may be an excellent fit.

## Responsibilities

Reporting to the Manager, the Multi-Platform Application Developer will perform the following duties:

- Develop new features and enhancements for the Expo-based mobile application
- Build and refine components, APIs, and pages within the Next.js web application
- Maintain and expand functionality in the Google Apps Script–based automation sheet
- Ensure smooth integration between all applications using Supabase as the unified backend
- Create and maintain tests to ensure that apps are running properly
- Troubleshoot issues across the mobile, web, and script environments
- Stay up-to-date with relevant technologies and patch vulnerabilities to mitigate risks
- Collaborate with other developers and work effectively within an existing codebase

## Requirements

- Bachelor’s degree in computer science, software development, or related field
- Strong experience with JavaScript and TypeScript
- Experience developing mobile apps with Expo or React Native
- Experience building scalable web applications with Next.js
- Familiarity with SQL and DBMS
- Familiarity with Version Control (Git, GitHub, etc.)
- Experience with Test-Driven Development and automated testing frameworks
- Strong understanding of UI/UX principles
- Ability to work independently or collaboratively within a team environment

## Nice-to-haves

- Hands-on experience working with Supabase (database, authentication, functions, and storage)
- Experience using Google Apps Script
